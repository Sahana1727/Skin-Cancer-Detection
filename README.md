#Skin Cancer Detection Using Deep Learning
##Project Description
This project is a web application that detects Skin cancer from skin lesion images usin depp learning. the application predicts whether the uploaded image is benign ar malignant
##Technologies Used
python
Flask
TensorFlow/Keras
HTML
CSS
JavaScript
##Dataset
HAM10000/ISIC Skin Cancer Dataset
##Feature
Upload Skin Lesion Images
Predict SKin Cancer type
User-friendly web interface
Deep learning-based image classification
##how to Run
Install python and required libraries
Install dependencies:
pip install -r requirements.txt
Run the application:
python app.py
open the application in your browser
